Today I mapped the first pass of the recommendation flow.

The goal is to make the site useful before the data model gets complicated. That means:

- asking for mood before genre
- showing fewer but stronger recommendations
- letting the user correct the recommendation quickly

Next I want to test whether the decision tree should be explicit or feel more like browsing.
